# Mini_SQL
MiniSQL

El proyecto MiniSQL cuanta con lexer que tiene una precision exacta sobre cualquier token que se quiera evaluar para poder funcionar el sql
Los errores que se manejan son los identificadores para no poder ser nuevosmayores a 31 evalua los errores de un string y toma en cuenta las
palabras reservadas en todo momento evaluando todo.
El unico error que maneja es el de no poder separar cuando no tiene un separador de por medio osea cuando todo viene junto
Programa no robusto.
Al guardar o generar el archivo .out debe de darle click en guardar y darle un nombre luego darle click en ok y listo 
lo guardo en el path que usted elije.
